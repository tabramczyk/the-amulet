# 09 - UI Structure

## Layout

```
+---------------------------------------------+
| TOP BAR                                      |
| Age: 23  |  Day: 2,555  |  Money: 142 gold  |
+----------------------+-----------------------+
| CLICK ACTIONS        | CONTINUOUS ACTIONS     |
| (Left Column)        | (Right Column)         |
|                      |                        |
| [Travel to Fields]   | * Begging (active)     |
| [Talk to Old Man]    |   Farming              |
| [Investigate Amulet] |   Train Concentration  |
|                      |                        |
+----------------------+-----------------------+
| BOTTOM PANEL                                 |
| Skills: Concentration 5 | Strength 12 | ... |
| Jobs: Beggar Lv.8 (XP: 234/500)            |
| Prestige: +8% Beggar | +3% Concentration    |
+---------------------------------------------+
```

## Panels

| Panel | Content | Update Frequency |
|-------|---------|-----------------|
| Top Bar | Age, date, money | Every tick |
| Click Actions | Context-dependent buttons | On location change / event |
| Continuous Actions | Available + active action | On location change |
| Bottom Panel | Skills, jobs, prestige | Every tick (levels) |

## Responsive Design
- Desktop: Two-column layout as shown
- Mobile: Stack columns vertically (click actions above continuous)
- Minimum width: 320px
