# Game Design Document - The Amulet

## One-Line Pitch
A fantasy idle RPG where time only moves when you act, and every short life feeds permanent reincarnation bonuses through a mysterious amulet.

## Quick Reference

- **Genre**: Idle / Clicker RPG
- **Platform**: Web browser (desktop + mobile)
- **Session Length**: ~1 hour per life
- **Core Loop**: Live -> Act -> Die -> Reincarnate -> Repeat (stronger)

## MVP Scope
- 3 locations (Slums, Fields, Village)
- 3 jobs (Beggar, Farmer, Laborer)
- 4 skills (Concentration, Strength, Intelligence, Endurance)
- 1 prestige layer
- 1 full life loop
