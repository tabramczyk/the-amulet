# 10 - MVP Scope

## Included in MVP

| System | Scope |
|--------|-------|
| Locations | 3: Slums, Fields, Village |
| Jobs | 3: Beggar, Farmer, Laborer |
| Skills | 4: Concentration, Strength, Intelligence, Endurance |
| Prestige | 1 layer (flat % bonus) |
| Life Loop | Full: age 16 -> death -> reincarnation |
| Story | Find amulet, amulet glows, touch to reincarnate |
| UI | Full layout with all panels |
| Save/Load | localStorage auto-save |

## NOT in MVP

| Feature | Reason |
|---------|--------|
| Multiple prestige layers | Complexity |
| Soft cap increases | Content scope |
| Achievements | Scope |
| Sound/Music | Scope |
| Mobile-specific UI | Can iterate later |
| Offline progression | Design says no background progression |
| Multiplayer / Leaderboards | Not planned |
| Backend / API | Static hosting only |
